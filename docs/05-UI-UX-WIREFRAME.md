# Dokumen UI/UX Wireframe

## Monitoring Website Pemerintah Provinsi Bali

---

## 1. Design Principles

### 1.1 Design Goals
- **Clarity**: Informasi status harus terlihat jelas dalam sekejap
- **Efficiency**: Akses cepat ke fitur yang sering digunakan
- **Responsiveness**: Dapat diakses dari desktop dan mobile
- **Accessibility**: Kontras warna yang baik, readable fonts

### 1.2 Color Scheme

```
Primary Colors:
┌────────────────────────────────────────────────────┐
│  #1E40AF  │  Primary Blue (Header, Buttons)       │
│  #3B82F6  │  Light Blue (Links, Accents)          │
└────────────────────────────────────────────────────┘

Status Colors:
┌────────────────────────────────────────────────────┐
│  #22C55E  │  Green - UP / Healthy                 │
│  #EF4444  │  Red - DOWN / Critical                │
│  #F59E0B  │  Yellow/Orange - Warning / Degraded   │
│  #6B7280  │  Gray - Unknown / Inactive            │
└────────────────────────────────────────────────────┘

Background Colors:
┌────────────────────────────────────────────────────┐
│  #F9FAFB  │  Light Gray (Page Background)         │
│  #FFFFFF  │  White (Cards, Panels)                │
│  #1F2937  │  Dark Gray (Sidebar)                  │
└────────────────────────────────────────────────────┘
```

### 1.3 Typography

```
Font Family: Inter (Sans-serif)

Hierarchy:
- H1: 24px / Bold   - Page Titles
- H2: 20px / Bold   - Section Headers
- H3: 16px / Medium - Card Titles
- Body: 14px        - Regular text
- Small: 12px       - Labels, captions
```
